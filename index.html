User Guide

Welcome to the Simple Calculator! Here’s how to use it:

  • Click number buttons (0–9) to enter values.
  • Use operator buttons (+, −, ×, ÷) for calculations.
  • Press = to see the result.
  • Press C to clear/reset the display.
  • Keyboard support: You can also type numbers and operators directly.
  • Error handling: Division by zero will show an error message.
